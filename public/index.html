Clone Wars

Code.Baert.io

As our final group project, we chose to clone codewars.com. We were given 2 weeks to complete our project. Codewars.com is a massive website with multiple aspects. We focused solely on the purpose of codewars: unit testing. This choice effected the content we showed in our views. For instance, the home page now is a users dashboard, omitting the social aspect of the clans.

My main responsibility was the server side unit tests of the users code. I used Docker to be able to run unsafe code without crashing the server. Codewars has an image which I used but it uses Mocha as its unit test framework.

GitHub

  • Date: January 2017
  • Production Time: 2.5 Weeks
  • Group size: 4

Maze Solver

Custom Maze Solver

Maze.Baert.io

This Maze Solver was made in response to a code challenge. It will solve a maze using Breadth-first search. It takes a step in every possible direction at the same time. then assumes the first path to find the end is the shortest.
